# sprig-game (a scrappy game engine before a game)
## Usage
### Sprig Web Editor
Requires `nodejs` & `npm`.
<br/> <br/>
- First, to install the dependencies, run `npm install`.
This will download all needed libraries to `node_modules/`.
- Next, to generate the ready-to-use code for the sprig website, run `npm run exportGame`. If successful, you will find that a `dist/bundle.js` has been generated. If unsuccessful, please create a new issue with the stdwarn & stderr.
- To use the code located in `dist/bundle.js`, select all and copy & paste to the sprig website's code editor.

### Local Development
Requires `nodejs` & `npm`
<br/><br/>
- First, to install the dependencies, run `npm install`.
  This will download all needed libraries to `node_modules/`.
- For a live development preview, run `npm run dev`.

### Prebuilt
No requirements
